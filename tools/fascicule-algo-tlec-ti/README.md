# Fascicule — Algorithmique & Programmation, Terminale TI (OnBuch)

Module 1 (Programmation) du programme MINESEC série TI : programmation web
(HTML, JavaScript, **PHP**, **MySQL**) et programmation procédurale en **langage C**.

## Contenu
- 12 chapitres (`ch01`…`ch12`) : web client, environnement, PHP, MySQL, C.
- Aide-mémoires PHP / SQL / C (`an_aidememoire.tex`).
- Sujets type Bac TI (`an_sujets.tex`) : 1 annale réelle (Bac Blanc Vogt 2025) + 5 sujets originaux corrigés.
- `preamble.tex` : design OnBuch + coloration syntaxique (listings) PHP/C/HTML/SQL/JS.
- `cover.tex` inclut `couverture.pdf` (couverture officielle).
- `Algorithmique-Programmation-TleTI-OnBuch.pdf` : PDF compilé (201 p.).
